# 🎮 Video Game Sales Dashboard

## 📌 Project Overview

This project analyzes video game sales data using Power BI to identify trends, top-performing games, popular platforms, and genre-wise performance.

---

## 🎯 Objective

* Analyze video game sales data
* Identify top-selling games
* Compare sales across regions
* Understand genre and platform performance

---

## 🛠 Tools Used

* Power BI
* Excel Dataset

---

## 📊 Dashboard Features

### 🔹 Page 1: Overview

* Total Sales Card
* Top 10 Games (Bar Chart)
* Sales by Genre (Pie Chart)
* Sales by Platform (Column Chart)
* Genre Filter (Slicer)

### 🔹 Page 2: Detailed Analysis

* Sales Trend (Line Chart)
* Publisher-wise Sales
* Region-wise Comparison
